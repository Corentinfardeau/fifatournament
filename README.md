## Usage
Clone repository on your computer. Run 
````
$ cd desktop
$ sudo npm install
$ sudo bower install
$ gulp serve
---
$ mongod
---
$ cd api
$ sudo npm install
$ sudo npm install -g nodemon --save
$ nodemon server.js
````

> You must have downloaded and installed MongoDB (http://docs.mongodb.org/manual/installation/)
